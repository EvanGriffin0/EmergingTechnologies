# Emerging Technologies Assignment

This repository contains solutions for the Emerging Technologies module assignment, focusing on implementing and analysing the Deutsch and Deutsch–Jozsa quantum algorithms using Qiskit.

## Repository Contents

The main file in this repository is:

- **problems.ipynb** – Jupyter notebook containing implementations and explanations for the assignment problems.

The notebook demonstrates:

- Generation of constant and balanced Boolean functions
- Classical determination of constant vs balanced functions
- Implementation of Deutsch's algorithm
- Implementation of the Deutsch–Jozsa algorithm for 4-bit functions
- Simulation of quantum circuits using Qiskit

## Requirements

Required Python packages are listed in `requirements.txt`. The notebook uses [Qiskit](https://qiskit.org/) for building and simulating quantum circuits.

## How to Run the Project

1. Clone the repository:

   ```bash
   git clone https://github.com/EvanGriffin0/EmergingTechnologies.git
   ```

2. Navigate into the repository:

   ```bash
   cd EmergingTechnologies
   ```

3. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

4. Run the notebook:

   ```bash
   jupyter notebook problems.ipynb
   ```
